# Data artifacts

This directory contains derived artifacts only. The original PDF corpus is not included.

- `corpus_metadata.csv`: metadata template for source papers.
- `source_links.csv`: source/DOI template.
- `extracted_json/`: LLM-extracted entities and relations for the 482-paper corpus.
- `graph_json/`: normalized graph-ready JSON records.
- `qa_evaluation/`: QA test set and evaluation inputs.

The raw parsed article text and original PDFs are intentionally excluded to reduce copyright and redistribution risk.
